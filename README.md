# EventFlow - Controle de Eventos

Sistema web do Grupo 8 para o Trabalho II de Programacao Web.

## Tema
Controle de Eventos.

## Tecnologias
- HTML
- CSS
- JavaScript
- LocalStorage

## Funcionalidades exigidas
- Home explicando o objetivo do sistema.
- Menu lateral com links principais.
- Cadastro e login de usuario.
- CRUD liberado apenas para usuario logado.
- Cadastro de eventos com campos obrigatorios e opcionais.
- Listagem em tabela mostrando N/A quando o campo opcional estiver vazio.
- Busca por nome, responsavel ou local.
- Edicao e exclusao de registros.
- Validacao para nao permitir eventos com nomes iguais.
- Armazenamento permanente no LocalStorage.
